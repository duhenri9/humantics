# COMANDOS PARA EXECUTAR NO SEU COMPUTADOR LOCAL

## 1. Primeiro, baixe o código do Replit

Opção A - Via Git (se tiver Git instalado):
```
git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio
```

Opção B - Download ZIP:
- No Replit, clique nos 3 pontos no canto superior direito
- Selecione "Download as ZIP"
- Extraia o ZIP no seu computador

## 2. Abra o Terminal/CMD na pasta do projeto

Windows:
- Abra a pasta extraída
- Segure Shift + Clique direito
- Selecione "Abrir janela do PowerShell aqui"

Mac/Linux:
- Abra o Terminal
- cd /caminho/para/pasta/extraida

## 3. Login no Docker Hub

```
docker login
```
Username: duhenri415
Password: [sua senha]

## 4. Build das imagens

FRONTEND:
```
docker build -f Dockerfile.frontend -t duhenri415/humantics-frontend:latest .
```

Aguarde terminar (pode demorar alguns minutos)...

BACKEND:
```
docker build -f Dockerfile.backend -t duhenri415/humantics-backend:latest .
```

## 5. Push para Docker Hub

FRONTEND:
```
docker push duhenri415/humantics-frontend:latest
```

BACKEND:
```
docker push duhenri415/humantics-backend:latest
```

## VERIFICAR SE DEU CERTO

Acesse:
- https://hub.docker.com/r/duhenri415/humantics-frontend
- https://hub.docker.com/r/duhenri415/humantics-backend

Deve aparecer "Last pushed: a few seconds ago"

## SE DER ERRO

Erros comuns:

1. "denied: requested access to the resource is denied"
   - Solução: Certifique-se de estar logado: docker login

2. "Cannot connect to the Docker daemon"
   - Solução: Abra o Docker Desktop primeiro

3. "no such file or directory"
   - Solução: Certifique-se de estar na pasta correta do projeto
